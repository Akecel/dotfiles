## Usage

### Clone on a new  machine

```
chezmoi init https://github.com/username/dotfiles.git
```

```
chezmoi diff
```

```
chezmoi apply
```

### Add new file

```
chezmoi add ~/.file
```

```
chezmoi -v apply
```
